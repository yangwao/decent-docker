# howto
### publictestnet
`docker pull yangwao/decent-docker:publictestnet-manual-1.2.2`

`docker-compose up publictestnet`

### mainnet - 1.2.2
`docker pull yangwao/decent-docker:production-manual4`

`docker-compose up mainnet`
